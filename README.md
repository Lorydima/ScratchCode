# ScratchCode Projects

## Introduction

This repository contains a collection of projects created with **Scratch** between **2019 and 2023**. These files represent exercises, experiments, and small games developed over the years to learn visual programming and basic programming logic.

Below is the list of available projects:

## Requirements

To open and run these projects, you need **Scratch 3**.

You can use:

* the online version: [https://scratch.mit.edu](https://scratch.mit.edu)
* or the official Scratch 3 offline editor

## Cloning the Repository
```bash
git clone https://github.com/Lorydima/ScratchCode.git
```

**How to Run the Projects**

1. Clone or download the repository
2. Open Scratch 3
3. Select **File → Load from your computer**
4. Open the desired `.sb3` file
